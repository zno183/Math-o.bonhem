Tableau des versions de Windows

Type Avantages Inconvénients
Windows 1.0 Outils inclus dans Windows : Paint, Notepad, Calculator et calendrier. Très lent, peu de logiciels compatibles, matériel limité, fenêtres non superposables.
Windows 2.0 Fenêtres superposables, raccourcis clavier, meilleures performances, interface plus agréable. Problèmes de sécurité, peu stable, peu de matériel compatible.
Windows 3.0 Interface moderne, meilleures performances, multitâche amélioré, meilleurs graphismes. Sécurité faible, configuration du matériel compliquée, consommation mémoire élevée.
OS/2 1.0 Système avancé, multitâche performant, plus stable que DOS. Très gourmand en ressources, compatibilité limitée, installation complexe.
Windows 3.1x Interface plus stable, support multimédia amélioré, police TrueType intégrée, grande bibliothèque de logiciels. Sécurité faible, multitâche limité, dépend encore de MS-DOS.
Windows NT 3.x Très stable, sécurité avancée, multitâche puissant, support de plusieurs processeurs. Très gourmand, peu adapté aux jeux, compatibilité limitée.
Windows 95 Introduction du bouton Démarrer, barre des tâches moderne, interface simple, support 32 bits, Plug and Play, meilleure intégration d’Internet. Stabilité moyenne, sécurité faible, mémoire limitée, pilotes parfois compliqués à installer.
Windows NT 4.0 Très stable, sécurité avancée, multitâche performant, interface moderne similaire à Windows 95, bon support réseau. Gourmand en ressources, peu compatible avec les jeux, support USB limité.
windows 98 Meilleure compatibilité matérielle, support USB amélioré, plus rapide que Windows 95, meilleure gestion d'Internet et des périphériques. Plantages fréquents, sécurité limitée, encore basé sur MS-DOS, performances faibles sur les vieux PC.
windows 2000 très stable Bonne sécurité Bon multitâche Meilleure gestion du matériel Performances fiables pour les entreprises Peu adapté aux jeux vidéo, Interface vieillissante aujourd'hui, Compatible avec moins de logiciels récents, performances faibles sur les vieux PC. Windows xp Très stable Interface agréable et simple à utiliser Compatible avec énormément de logiciels et de jeux Bonnes performances Très populaire auprès du grand public et des entreprises Ancien système aujourd'hui Vulnérable aux virus après la fin du support Microsoft Sécurité dépassée Compatible avec moins de logiciels récents Certaines anciennes versions étaient gourmandes en ressources pour l'époque
Windows vista Interface moderne avec Aero Sécurité améliorée Meilleure recherche de fichiers Support multimédia amélioré Nouveaux effets graphiques Très gourmand en ressources Lent sur beaucoup de PC de l'époque Problèmes de compatibilité avec certains logiciels et pilotes Nombreux messages de sécurité agaçants Mauvaise réputation auprès des utilisateurs
